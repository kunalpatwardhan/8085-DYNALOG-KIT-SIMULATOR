READY

 

      It is an input active high pin.

      If a high signal is present on this pin when the microprocessor is performing a read or a write operation then it indicates that the memory or peripheral device is ready to send or receive to or from the microprocessor.

      If the signal is low peripheral is not ready for the data transfer and the microprocessor will keep on waiting till the peripheral is ready i.e. till the signal on the READY pin is high.

      This is done to synchronize the data transfer between the microprocessor and the device.

 

 

                                                  

      It is also known as HANDSHAKING SIGNAL.
