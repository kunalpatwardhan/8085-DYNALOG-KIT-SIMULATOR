STACK POINTER

 

      It is a 16-bit register.

      It is used as a memory pointer.

      It points to a memory location in the R/W (READ/WRITE) memory, called the stack.

      The beginning of the STACK is defined by loading a 16-bit address in the stack pointer. Usually the end locations of the R/W memory are allotted to the stack.

      Every memory location in the stack can be used as an 8-bit register and can be used to store an 8-bit data.
