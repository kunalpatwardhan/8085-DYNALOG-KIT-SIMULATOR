MICROPROCESSOR-INITIATED OPERATIONS & 8085 BUS ORGANISATION

 

      The MPU performs primarily four operations

 

v     MEMORY READ:  Reads data (or instructions) from memory.

v     MEMORY WRITE:  Writes data (or instructions) into memory.

v     I/O READ:  Accepts data from input devices.

v     I/O WRITE:  Sends data to output devices.

 

      To communicate with a peripheral (or a memory location), the MPU needs to perform the following steps:

           STEP 1: Identify the peripheral or the memory location (with its address).

           STEP 2: Transfer binary information (data and instructions).

           STEP 3: Provide timing or synchronization signals.

 

      The 8085 MPU performs these functions using three sets of communication lines called buses:

                  *THE ADDRESS BUS.

             *THE DATA BUS.

             *THE CONTROL BUS.
