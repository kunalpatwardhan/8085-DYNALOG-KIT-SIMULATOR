PROGRAM COUNTER (PC)

 

      It is a 16-bit register used to hold addresses. This is so because the size of the addresses is 16-bits.

      The microprocessor uses this register to sequence the execution of the instructions.

      Its function is to point to the memory location from which the next byte is to be fetched. When the byte is being fetched the program counter is incremented by one (or sometimes two in case of JUMP) instructions so that it points to the next instruction.
