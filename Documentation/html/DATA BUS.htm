DATA BUS (AD7 – AD0)

 

      The data bus is a group of eight lines used for data flow.

      These lines are bi-directional – data flow in both directions between the MPU and the memory and the peripheral devices.

      The MPU uses the data bus to perform the second function -- transferring binary information.

      The 8 data lines enable the MPU to manipulate 8-bit data ranging from 00 to FF (28= 256 numbers). The largest number that can appear on the data bus is 11111111.
