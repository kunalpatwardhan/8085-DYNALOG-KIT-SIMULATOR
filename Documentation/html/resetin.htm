It is an input active low signal.

      A logic 0 signal applied to this pin resets the microprocessor and the PROGRAM COUNTER is cleared.

      Resets the INTERRUPT ENABLE (INTA) and HOLD ACKNOWLEDGE (HLDA) flip-flop after tristating the DATA BUS, ADDRESS BUS AND THE CONTROL BUS.

      The microprocessors internal registers are altered with unpredictable results.

      There are two types of RESETS

    1. POWER ON RESET
    2. MANUAL RESET

      For proper manual reset operation the switch should be pressed for at least three-clock pulse duration.
