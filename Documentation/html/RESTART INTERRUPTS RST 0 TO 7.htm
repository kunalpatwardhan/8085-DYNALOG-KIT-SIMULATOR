RESTART INTERRUPTS RST 0 TO 7

 

      The 8085 instruction set includes eight RST (restart instructions)

      They are 1-byte CALL instructions that transfer the program execution to a specific location on page 00H

      They are maskable vectored interrupts.

      They are externally initiated hardware interrupts.
