ABOUT THE PROJECT

 

It is an 8085 – simulator based on the 8085 DYNALOG KIT programmed in VISUAL BASIC. Brief information about the 8085 microprocessor has also been given. All the theory that has been documented has been done using standard reference books.

 

 

WHAT INSPIRED US?

 

We as students faced problem studying and gathering information from a number of reference books. Also we did not have anything to test the programmes written by us. Hence we thought of doing this project which could help anybody trying to study 8085 up.

 

 

SALIENT FEATURES

 

·       No part of this program has been copied from anywhere. All the programming has been done from scratch using our own logic and understanding.

·       Only standard reference books have been used for the documentation.

·       It’s a real time simulator unlike any other 8085 simulator available in the market.

·       It gives actual laboratory experience.

·       And finally it’s a project done by the students for the students.
